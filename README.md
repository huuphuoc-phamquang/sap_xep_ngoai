# sap_xep_ngoai
Chương trình này được xây dựng bằng C++ để minh họa trực quan cách hoạt động của thuật toán External Sort (Sắp xếp ngoại). Đây là kỹ thuật được ứng dụng trong thực tế khi cần sắp xếp các tập tin dữ liệu khổng lồ (hàng chục hoặc hàng trăm GB) vượt quá dung lượng bộ nhớ RAM của máy tính.
